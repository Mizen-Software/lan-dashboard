## Installation
Open the terminal and navigate to the location you want to install this.
```
virtualenv lan-dashboard
cd lan-dashboard
Scripts\activate
```
copy .gitignore into the directory

```
git init
git remote add origin https://github.com/Mizen-Software/lan-dashboard.git
git pull origin master
pip install -r requirements.txt
```
